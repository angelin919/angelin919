# Нагрузочное тестирование интернет-магазина "Читай-город"

## 📋 Описание проекта
Проект по нагрузочному тестированию сайта [www.chitai-gorod.ru](https://disk.yandex.ru/d/Rc02HJ0EmeclPQ) с помощью Apache JMeter. 
Тест имитирует поведение реального пользователя, проходящего по типовому сценарию: главная → каталог → категория → подкатегория → карточка книги.

## 🛠 Инструменты
- Apache JMeter

## ⚙️ Настройки теста
- **Количество виртуальных пользователей**: 3
- **Временные ожидания**: Uniform Random Timer (1000-3000 мс)
- **Распределение нагрузки**: Throughput Controller (40%/40%)
- **Количество итераций**: 1
