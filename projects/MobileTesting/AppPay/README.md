# 💳 AppPay – Тестирование мобильного приложения для бесконтактных платежей (Android)

Этот проект содержит мой вклад как **тестировщика** в командное тестирование мобильного приложения для бесконтактных оплат.
Приложение позволяет добавлять и удалять банковские карты, совершать платежи, а также переключать светлую и тёмную темы интерфейса.

## 📱 О проекте
- **Платформа:** Android
- **Основной протестированный функционал:**
  - Управление картами (добавление, удаление, переименование, валидация)
  - Переключение светлой/тёмной темы

## 📲 Тестовое окружение
- **Устройство:** Xiaomi Poco M5 (Android 12 физическое устройство)
- **Инструменты:** TestIt, ADB для сбора логов

## 🛠 Инструменты и методы
- **Тест-менеджмент:** тест-кейсы, чек-листы
- **Баг-трекинг:** Exsel
- **Дополнительно:** команды ADB, анализ логов
